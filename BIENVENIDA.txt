═══════════════════════════════════════════════════════════════════════════════
                          💡 CONSEJOS ÚTILES
═══════════════════════════════════════════════════════════════════════════════

1. Las credenciales de Firebase NO funcionan sin actualizar
   └─ Abre js/firebase-config.js y reemplaza todo

2. La tienda usa localStorage para guardar el carrito
   └─ Funciona en el mismo navegador
   └─ Se pierde si limpias datos
   └─ Perfecto para pruebas

3. Los impuestos son 11.5% por defecto (IVU Puerto Rico)
   └─ Para cambiar: edita js/cart.js línea ~107

4. Las imágenes deben ser HTTPS
   └─ URLs de http:// no funcionarán

5. Firestore Rules son IMPORTANTES para seguridad
   └─ Sin ellas cualquiera puede modificar productos

6. Prueba TODO en navegación privada primero
   └─ localStorage funciona diferente allí
   └─ Así sabes si hay problemas reales

7. Mantén copias de seguridad de tus datos
   └─ Descarga JSON regularmente desde Firebase
