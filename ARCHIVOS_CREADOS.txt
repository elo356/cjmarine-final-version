🚀 FLUJO DE ACTIVACIÓN:
═════════════════════════════════════════════════════════

PASO 1: LEER
   └─ Lee INICIO_RAPIDO.md (5 minutos)

PASO 2: CONFIGURAR
   └─ Actualiza js/firebase-config.js con tus credenciales

PASO 3: CARGAR DATOS
   └─ Crea colección "products" en Firebase
   └─ Agrega tus productos (mínimo: name, price, stock)

PASO 4: PROBAR
   └─ Abre store.html en navegador
   └─ Busca, filtra, agrega al carrito
   └─ Completa compra de prueba

PASO 5: PERSONALIZAR (opcional)
   └─ Edita css/store.css para cambiar colores
   └─ Edita impuesto en js/cart.js si es necesario


💾 BASE DE DATOS FIREBASE:
═════════════════════════════════════════════════════════

LECTURA AUTOMÁTICA: products
├─ name (texto, requerido)
├─ price (número, requerido)
├─ stock (número, requerido)
├─ sku (texto)
├─ category (texto)
├─ description (texto)
├─ image (URL HTTPS)
└─ ... otros campos personalizados

ESCRITURA AUTOMÁTICA: orders
└─ Se crea al completar compra con:
   ├─ items (carrito)
   ├─ email (cliente)
   ├─ phone (cliente)
   ├─ subtotal, taxes, total
   └─ createdAt (timestamp)

🔐 REGLAS FIRESTORE RECOMENDADAS:
═════════════════════════════════════════════════════════

rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {
    match /products/{document=**} {
      allow read;
      allow write: if false;
    }
    match /orders/{document=**} {
      allow create;
      allow read, update, delete: if false;
    }
  }
}
